### implementation
+ Gradle
```
compile 'jsc.kit.wheel:wheel-view:_latestVersion'
```
+ Maven
```
<dependency>
  <groupId>jsc.kit.wheel</groupId>
  <artifactId>wheel-view</artifactId>
  <version>_latestVersion</version>
  <type>pom</type>
</dependency>
```
